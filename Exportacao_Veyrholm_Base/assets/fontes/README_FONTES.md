# Fontes

O projeto atual nao possui arquivos de fonte externos.

Fontes usadas via CSS:

- Interface: `"Trebuchet MS", "Segoe UI", system-ui, sans-serif`
- Titulos: `Georgia, "Times New Roman", serif`

Para empacotar uma identidade propria em outro RPG, adicione arquivos `.woff2` nesta pasta e declare `@font-face` em `assets/estilos/styles.css`.
