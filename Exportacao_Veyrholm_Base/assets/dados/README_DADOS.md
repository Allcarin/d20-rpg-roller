# Dados exportados

- `ashesOfVeyrholm.js`: campanha canonica atual como fonte primaria de dados.
- `validateCampaign.js`: validador do grafo de cenas e integridade basica.
- `PLAYBOOK.md`: regras canonicas do universo Cinzas de Veyrholm.

Para outros RPGs, use `ashesOfVeyrholm.js` como exemplo tecnico, mas substitua lore, personagens, locais, faccoes, itens, cenas e flags.
